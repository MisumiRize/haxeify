# require-from-haxe

This sample performs typed `require` by haxeify and Haxe (>= 3.2.0).

## Requirements

* Haxe (>= 3.2.0)

## Installation

With [npm](https://npmjs.org/):

```bash
$ npm install
```

## Run

Compile with npm:

```bash
$ npm run build
```

You can see result by running with node:

```bash
$ node bundle.js
```
